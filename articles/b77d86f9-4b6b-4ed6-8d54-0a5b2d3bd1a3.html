Less Noise, Same Certificate: Retain Sensitivity for Unlearning

核心贡献

定义了保留敏感度概念，指出遗忘不需要保护保留数据的隐私，因此可以使用比DP噪声更小的校准，同时保持相同的认证保证

方法概述

将遗忘的噪声校准从全局敏感度改为保留敏感度——即在固定保留数据集R的情况下，删除集U变化时的最坏输出变化。理论上证明了这种方法噪声更小且证书相同，并在最小生成树、PCA和ERM等任务上进行了验证

关键结论

保留敏感度虽然不足以用于差分隐私，但对于遗忘恰好充分；通过利用保留数据R的规律性，可以进一步减少噪声并提升效用

原文摘录

Certified machine unlearning aims to provably remove the influence of a deletion set $U$ from a model trained on a dataset $S$, by producing an unlearned output that is statistically indistinguishable from retraining on the retain set $R:=S\setminus U$. Many existing certified unlearning methods adapt techniques from Differential Privacy (DP) and add noise calibrated to global sensitivity, i.e., the worst-case output change over all adjacent datasets. We show that this DP-style calibration is often overly conservative for unlearning, based on a key observation: certified unlearning, by definition, does not require protecting the privacy of the retained data $R$. Motivated by this distinction, we define retain sensitivity as the worst-case output change over deletions $U$ while keeping $R$ fixed.
